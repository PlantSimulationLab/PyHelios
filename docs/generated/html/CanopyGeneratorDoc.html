Canopy Generator Plugin

The Canopy Generator plugin provides functionality for creating plant canopies and vegetation assemblies for various species.

Overview

The Canopy Generator plugin enables:

  • Species-specific canopy generation
  • Customizable plant spacing and arrangement
  • Integration with other modeling plugins
  • Support for multiple vegetation types

Basic Usage

from pyhelios import Context
from pyhelios.plugins import CanopyGenerator
# Create context and canopy generator
context = Context()
canopy_gen = CanopyGenerator(context)
# Generate a simple canopy
canopy_id = canopy_gen.generate_canopy(
species="grass",
area_bounds=[(0, 0), (10, 10)],
plant_density=100 # plants per square meter
)

Species Support

The plugin supports various plant types:

  • Grasses and ground cover
  • Shrubs and bushes
  • Agricultural crops
  • Forest understory vegetation

Configuration Options

Plant Density

# Uniform density
canopy_gen.set_uniform_density(50) # plants/mÂ²
# Variable density with spatial patterns
canopy_gen.set_density_map("density_pattern.txt")

Spatial Arrangement

# Regular grid pattern
canopy_gen.set_arrangement("grid", spacing=0.5)
# Random distribution
canopy_gen.set_arrangement("random", min_distance=0.2)
# Clustered patterns
canopy_gen.set_arrangement("clustered", cluster_size=5, cluster_separation=2.0)

Integration with Other Plugins

With Weber-Penn Trees

from pyhelios import WeberPennTree, WPTType
# Generate background canopy
canopy_id = canopy_gen.generate_canopy("grass", area_bounds=[(0, 0), (20, 20)])
# Add individual trees
wpt = WeberPennTree(context)
tree_positions = [(5, 5), (10, 10), (15, 15)]
for pos in tree_positions:
tree_id = wpt.build_tree(
tree_type=WPTType.OLIVE,
position=pos
)

With Radiation Modeling

from pyhelios import RadiationModel
# Generate canopy
canopy_id = canopy_gen.generate_canopy("corn", area_bounds=[(0, 0), (50, 50)])
# Run radiation simulation on canopy
with RadiationModel(context) as radiation:
radiation.add_radiation_band("PAR")
radiation.set_direct_ray_count("PAR", 1000)
radiation.run_band("PAR")

Advanced Features

Custom Species Definition

# Define custom plant species
custom_species = {
"name": "custom_crop",
"leaf_area_index": 3.5,
"plant_height": 0.8,
"leaf_angle_distribution": "spherical",
"leaf_size_range": (0.05, 0.15)
}
canopy_gen.register_species(custom_species)
canopy_id = canopy_gen.generate_canopy("custom_crop", area_bounds=[(0, 0), (10, 10)])

Temporal Development

# Simulate canopy growth over time
initial_canopy = canopy_gen.generate_canopy(
"wheat",
area_bounds=[(0, 0), (30, 30)],
growth_stage="seedling"
)
# Update growth stage
canopy_gen.update_growth_stage(initial_canopy, "flowering")
canopy_gen.update_growth_stage(initial_canopy, "mature")

Performance Considerations

# For large canopies, use LOD (Level of Detail)
canopy_gen.set_level_of_detail(
distance_thresholds=[10, 50, 100],
detail_levels=["high", "medium", "low"]
)
# Optimize memory usage
canopy_gen.set_memory_optimization(True)
canopy_gen.set_max_primitives_per_plant(50)
